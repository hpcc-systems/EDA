Index-based DISTRIBUTE

DISTRIBUTE(recordset, index [, joincondition ] )

This form redistributes the recordset based on the existing distribution of the specified index, where the linkage between the two is determined by the joincondition. Records for which the joncondition is true will end up on the same node.
