In-line Record Definitions

Curly braces ({}) are lexical equivalents to the keywords RECORD and END that can be used anywhere RECORD and END are appropriate. Either form (RECORD/END or {}) can be used to create “on-the-fly” record formats within those functions that require record structures (OUTPUT, TABLE, DATASET etc.), instead of defining the record as a separate attribute.
