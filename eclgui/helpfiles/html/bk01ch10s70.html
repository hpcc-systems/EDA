LOG

LOG(n)

nThe real number to evaluate.
Return:LOG returns a single real value.

The LOG function returns the base-10 logarithm of the parameter.

Example:

MyLogPI := LOG(3.14159); //0.49715

See Also: EXP, SQRT, POWER, LN
