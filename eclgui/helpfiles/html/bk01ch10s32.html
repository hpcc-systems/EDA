EBCDIC

EBCDIC(recordset)

recordsetThe set of records to process. This may be the name of a dataset or a record set derived from some filter condition, or any expression that results in a derived record set.
Return:EBCDIC returns a set of records

..

The EBCDIC function returns the recordset with all STRING fields translated from ASCII to EBCDIC.

Example:

EBCDICRecs := EBCDIC(SomeASCIIInput);
