Attributes as Actions

Any Attribute that defines a single value of some sort (that is, it does not define a record set) can be executed as an Action. These same actions can be executed by submitting the names of the Attributes as queries, like this:

Attr1; //These all generate the same result values
Attr2; // as the previous examples
Attr3;
