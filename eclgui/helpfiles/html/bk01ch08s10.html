SHARED

SHARED definition

definition An Attribute definition.

The SHARED keyword indicates an Attribute that is available for use throughout the module (i.e. module scope). Without either the SHARED or EXPORT keywords, an Attribute's scope is limited to the next SHARED or EXPORTed Attribute

Example:

BadPeople := Person(EXISTS(trades(EXISTS(phr(phr_rate > '4'))));
        //local only to the GoodHouses Attribute
SHARED GoodHouses := Household(~EXISTS(BadPeople));
        //available all thru the module

See Also: EXPORT, Attribute Visibility, MODULE
