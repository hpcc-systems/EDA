Actions as Attributes

Conversely, by simply giving any Action an Attribute name you turn it into an Attribute definition, therefore no longer a directly executable action. For example,

OUTPUT(Person);

is an action, but

Attr4 := OUTPUT(Person);

is an attribute definition and does not immediately execute when submitted as part of a query. To execute the action inherent in the attribute, you must execute the Attribute name you've given to the Action, like this:

Attr4; // run the OUTPUT(Person) action
