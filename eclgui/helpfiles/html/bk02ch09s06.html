GetExpandLogicalFileName

ThorLib.GetExpandLogicalFileName ( filename )

filenameA null-terminated string containing the logical name of the file.
Return: GetExpandLogicalFileName returns a VARSTRING (null-terminated) value.

The GetExpandLogicalFileName function returns a string containing the expanded logical filename (ncluding the default scope, if the filename does not contain a leading tilde), all in lowercase. This is the same value as is used internally by DATASET and OUTPUT.

Example:

A := ThorLib.GetExpandLogicalFileName('Fred');
