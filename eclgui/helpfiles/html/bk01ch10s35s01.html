EVALUATE action

[attrname := ] EVALUATE(expression) ;

attrnameOptional. The action name, which turns the action into an attribute definition, therefore not executed until the attrname is used as an action.
expressionThe function to call in an action context.

The EVALUATE action names an expression (typically a function call) to execute in an action context. This is mainly useful for calling functions that have side-effects, where you don't care about the return value.

Example:

myService := SERVICE
 UNSIGNED4 doSomething(STRING text);
END;

ds := DATASET('MyFile', {STRING20 text} , THOR);

APPLY(ds, EVALUATE(doSomething(ds.text)));
 //calls the doSomething function once for each record in the ds
 // dataset, ignoring the returned values from the function

See Also: APPLY, SERVICE
