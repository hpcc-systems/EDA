Attribute Visibility

ECL Attributes are organized into modules. Within a module, you may define as many Attributes as needed. An IMPORT statement (see the IMPORT keyword) identifies any other modules whose visible Attributes will be available for use in the current attribute definition.

IMPORT AnotherModule; //imports attributes from AnotherModule
Attribute1 := 5;     //as many definitions as needed

The following fundamental attribute visibility scopes are available in ECL: "Global," Module, and Local.
