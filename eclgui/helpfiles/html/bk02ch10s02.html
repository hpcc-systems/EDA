Target

STD.System.Thorlib.Target ( )

Return: Target returns a VARSTRING value.

The Target function returns the name of the cluster running the workunit. Not supported on Roxie clusters. This name is used by ECLplus.exe to specify the the target cluster for a workunit.

Example:

A := STD.System.Thorlib.Target();
