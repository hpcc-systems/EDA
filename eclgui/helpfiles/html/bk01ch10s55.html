HAVING

HAVING( groupdataset, expression )

groupdatasetThe name of a GROUPed record set.
expressionThe logical expression by which to filter the groups.
Return:HAVING returns a GROUPed record set.

The HAVING function returns a GROUPed record set containing just those groups for which the expression is true. This is similar to the HAVING clause in SQL. Available for use only in hthor and Roxie.

Example:

MyGroups := GROUP(SORT(Person,lastname),lastname);
     //group by last name
Filtered := HAVING(MyGroups,COUNT(ROWS(LEFT)) > 10);
     //filter out the small groups

See Also: GROUP
