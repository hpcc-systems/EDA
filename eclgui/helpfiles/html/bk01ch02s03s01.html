Attribute Name Rules

Attribute names begin with a letter and may contain only letters, numbers, or underscores (_).

My_First_Attribute1 := 5; // valid name
My First Attribute := 5;  // INVALID name, spaces not allowed

You may name an attribute with the name of a previously created module in the ECL Repository, if the attribute is defined with an explicit ValueType.
