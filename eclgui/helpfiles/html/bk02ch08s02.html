Year

STD.Date.Year( date )

dateA date value in the Date_t format.
Return:Year returns an INTEGER value.

The Year function returns the Year number from the date value.

Example:

IMPORT STD;
UNSIGNED4 MyDate := 20120101;   //January 1, 2012
  
Y := STD.Date.Year(MyDate);
    //Y contains 2012
