Value Attributes

A Value Attribute is defined as any Attribute whose expression is an arithmetic or string expression with a single-valued result. For example, the following are all Value Attributes:

ValueTrue      := 1;
FloridianCount := COUNT(Person(Person.per_st = 'FL'));
OldAgeSum     := SUM(Person(Person.Age >= 65),Person.Age);
