ClearSuperFile

STD.File.ClearSuperFile( superfile, [ , delete ] )

superfileA null-terminated string containing the logical name of the superfile.
deleteA boolean flag specifying whether to delete the sub-files from disk or just remove them from the superfile list of files. If omitted, the default is to just remove them from the superfile list of files.
Return: Null.

The ClearSuperFile function removes all sub-files from the list of files comprising the superfile.

This function may be included in a superfile transaction.

Example:

SEQUENTIAL(
 STD.File.StartSuperFileTransaction(),
 STD.File.ClearSuperFile('MySuperFile'),
 STD.File.FinishSuperFileTransaction()
);
