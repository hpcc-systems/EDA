EXPORT

EXPORT definition

definition A definition.

The EXPORT keyword explicitly allows other definitions to import the specified definition. Definitions not explicitly flagged for export are local to the folder within which they reside. Without either the SHARED or EXPORT keywords, a definitions's scope is limited to the next SHARED or EXPORT.

Example:

EXPORT MyAttribute := 5;

// allows other definitions to use MyModule.MyAttribute if they import
// MyModule

See Also: IMPORT, SHARED, Attribute Visibility, MODULE
