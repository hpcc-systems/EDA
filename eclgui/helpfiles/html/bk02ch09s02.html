Nodes

STD.System.Thorlib.Nodes( )

Return: Nodes returns an UNSIGNED INTEGER4 value.

The Nodes function returns the number of nodes in the cluster. This number is the same as the CLUSTERSIZE compile time constant. The Nodes function is evaluated each time it is called, so the choice to use the function versus the constant depends upon the circumstances.

Example:

A := STD.System.Thorlib.Nodes();
