PULL

PULL(dataset)

datasetThe set of records to fully load into the Data Refinery.
Return:PULL returns a recordset.

The PULL function is a meta-operation intended only to hint that the dataset should be fully loaded into the Data Refinery before continuing the operation in Data Refinery. This allows the Data Refinery optimizer to ensure optimal performance and frees up the Complex Analysis Engine sooner.

Example:

MySet := PULL(Person);
     //load Person into Data Refinery before continuing
