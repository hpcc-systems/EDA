Boolean Attributes

A Boolean Attribute is defined as any Attribute whose definition is a logical expression resulting in a TRUE/FALSE result. For example, the following are all Boolean Attributes:

IsBoolTrue  := TRUE;
IsFloridian := Person.per_st = 'FL';
IsOldPerson := Person.Age >= 65;
